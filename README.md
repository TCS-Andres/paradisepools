# Paradise Pools & Pressure Cleaning — Preliminary Homepage

A single-page, responsive marketing site for **Paradise Pools & Pressure Cleaning**, a family-run pool service and pressure-washing company serving Broward and Miami-Dade County, Florida.

This is a **speculative / preliminary concept** built to open a conversation with the business owner.

## Stack
- Plain HTML + CSS + JS (no framework)
- Tailwind CSS via CDN
- Lucide icons via CDN
- Google Fonts: Inter (body) + Poppins (display)

## Run it
Just double-click `index.html` — no build step required.
